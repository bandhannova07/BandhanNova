Introduction to Flutter

Build Beautiful Cross-Platform Mobile Apps

What is Flutter?

Flutter is Google's revolutionary UI toolkit for building natively compiled applications for mobile, web, and desktop from a single codebase. Launched in 2017, Flutter has transformed mobile app development by enabling developers to create high-performance, visually attractive applications that run seamlessly on both iOS and Android platforms using just one programming language: Dart.

What sets Flutter apart is its unique architecture and rendering engine. Unlike other cross-platform frameworks that rely on web views or native components, Flutter draws its own UI using a custom rendering engine called Skia. This approach ensures consistent performance and appearance across all platforms, eliminating the "write once, debug everywhere" problem that plagues many cross-platform solutions.

Flutter's widget-based architecture makes building complex UIs intuitive and maintainable. Everything in Flutter is a widget - from layout structures to interactive elements. This compositional approach allows developers to create custom, pixel-perfect designs while maintaining excellent performance. Major companies like Google Pay, Alibaba, BMW, and Nubank have adopted Flutter for their production apps, demonstrating its enterprise-ready capabilities.

Your First Flutter App

main.dart 
// Welcome to Flutter Development!
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'My First Flutter App',
      theme: ThemeData(
        primarySwatch: Colors.blue,
        visualDensity: VisualDensity.adaptivePlatformDensity,
      ),
      home: WelcomeScreen(),
    );
  }
}

class WelcomeScreen extends StatefulWidget {
  @override
  _WelcomeScreenState createState() => _WelcomeScreenState();
}

class _WelcomeScreenState extends State {
  String studentName = "Flutter Developer";
  int appsBuilt = 0;
  bool isLearning = true;

  void incrementApps() {
    setState(() {
      appsBuilt++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Welcome to Flutter'),
        backgroundColor: Colors.blue,
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Icon(
              Icons.flutter_dash,
              size: 100,
              color: Colors.blue,
            ),
            SizedBox(height: 20),
            Text(
              'Hello, $studentName!',
              style: TextStyle(
                fontSize: 24,
                fontWeight: FontWeight.bold,
                color: Colors.blue[800],
              ),
            ),
            SizedBox(height: 10),
            Text(
              'Apps Built: $appsBuilt',
              style: TextStyle(fontSize: 18),
            ),
            SizedBox(height: 10),
            Text(
              isLearning ? 'Currently Learning Flutter' : 'Flutter Expert',
              style: TextStyle(
                fontSize: 16,
                color: Colors.green[600],
              ),
            ),
            SizedBox(height: 30),
            ElevatedButton(
              onPressed: incrementApps,
              child: Text('Build Another App'),
              style: ElevatedButton.styleFrom(
                padding: EdgeInsets.symmetric(horizontal: 30, vertical: 15),
              ),
            ),
          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () {
          ScaffoldMessenger.of(context).showSnackBar(
            SnackBar(content: Text('Welcome to Flutter Development!')),
          );
        },
        child: Icon(Icons.favorite),
      ),
    );
  }
}

Flutter is Google's UI toolkit for building beautiful, natively compiled applications for mobile, web, and desktop from a single codebase. It uses the Dart programming language and provides a rich set of pre-designed widgets.

Flutter enables cross-platform development from a single codebase

Why Choose Flutter?

Flutter allows you to write once and deploy everywhere. Popular apps like Google Ads, Alibaba, and BMW use Flutter for their mobile applications.

Flutter Architecture

Flutter's architecture consists of three main layers:

  • Framework Layer: Widgets, rendering, and animation APIs written in Dart
  • Engine Layer: Core C++ engine handling graphics, text layout, and Dart runtime
  • Platform Layer: Platform-specific embedders for iOS, Android, web, and desktop
