Chapter 1: JavaScript Fundamentals

60 minutes Beginner

Welcome to JavaScript!

JavaScript is the programming language of the web. It's what makes websites interactive, dynamic, and engaging. From simple button clicks to complex web applications, JavaScript powers the modern internet.

Why Learn JavaScript?

JavaScript is the most popular programming language in the world, used by 97% of websites. It's essential for frontend development and increasingly popular for backend development with Node.js.

Your First JavaScript Code

Let's start with the traditional "Hello, World!" program. This simple example will introduce you to JavaScript syntax.

Hello World in JavaScript 
// This is a comment - it doesn't run as code
console.log("Hello, World!");

// You can also use alert to show a popup
alert("Welcome to JavaScript!");

// Variables store data
let message = "JavaScript is awesome!";
console.log(message);

JavaScript in Web Pages

JavaScript can be included in HTML pages in several ways:

  • Inline: Directly in HTML attributes (not recommended)
  • Internal: Inside <script> tags in the HTML document
  • External: In separate .js files (best practice)
Including JavaScript in HTML 
<!DOCTYPE html>
<html>
<head>
    <title>My JavaScript Page</title>
</head>
<body>
    <h1 id="heading">Welcome!</h1>
    <button onclick="changeText()">Click Me!</button>
    
    <script>
        function changeText() {
            document.getElementById('heading').textContent = 'Hello JavaScript!';
        }
    </script>
</body>
</html>
