JavaScript Fundamentals

Understanding the Building Blocks of Web Interactivity

What is JavaScript?

JavaScript is the programming language that brings websites to life. Created in 1995 by Brendan Eich, it has evolved from a simple scripting language to the backbone of modern web development. Today, JavaScript powers everything from interactive buttons to complex web applications like Gmail, Facebook, and Netflix.

Unlike HTML (which structures content) and CSS (which styles it), JavaScript adds behavior and interactivity. It's what makes dropdown menus work, forms validate input, and pages update without refreshing. JavaScript runs directly in your web browser, making it instantly accessible to billions of users worldwide.

What makes JavaScript special is its versatility. It's not just for websites anymore - you can use it to build mobile apps, desktop applications, server-side programs, and even control IoT devices. This "write once, run everywhere" philosophy has made JavaScript the most popular programming language in the world.

Your First JavaScript Program

hello-world.js 
// Welcome to JavaScript!
// This is a comment - it explains what the code does

// Display a message in the browser console
console.log("Hello, World!");

// Show a popup message to the user
alert("Welcome to JavaScript programming!");

// Store information in variables
let studentName = "Alex";
let courseTitle = "JavaScript Fundamentals";
let currentYear = 2024;

// Combine variables to create a personalized message
let welcomeMessage = `Hi ${studentName}! Welcome to ${courseTitle} in ${currentYear}.`;
console.log(welcomeMessage);

// Simple function to calculate age
function calculateAge(birthYear) {
    return currentYear - birthYear;
}

// Use the function
let age = calculateAge(2000);
console.log(`You are ${age} years old!`);

Variables and Data Types

Storing and Working with Different Types of Information

Understanding Variables

Variables are like containers that store data values. Think of them as labeled boxes where you can put different types of information - numbers, text, true/false values, and more. In JavaScript, variables are essential for creating dynamic programs that can remember information and perform calculations.

Modern JavaScript provides three ways to declare variables: let, const, and var. Each has different rules about when and how they can be used. Understanding these differences is crucial for writing clean, bug-free code.

JavaScript is a "dynamically typed" language, which means you don't need to specify what type of data a variable will hold. The JavaScript engine automatically determines the data type based on the value you assign. This flexibility makes JavaScript easy to learn but requires understanding of how different data types behave.

Variable Declaration and Data Types

variables-and-types.js 
// Variable Declaration Methods

// let - for variables that can change
let userName = "Sarah";
let userAge = 28;
let isLoggedIn = false;

// const - for values that won't change
const PI = 3.14159;
const siteName = "BandhanNova";
const maxUsers = 1000;

// var - older way (avoid in modern code)
var oldStyleVariable = "This is outdated";

// Data Types in JavaScript

// 1. String (text)
let firstName = "John";
let lastName = 'Doe';  // Single or double quotes work
let fullName = `${firstName} ${lastName}`;  // Template literals

// 2. Number (integers and decimals)
let age = 25;
let price = 99.99;
let temperature = -5;

// 3. Boolean (true/false)
let isStudent = true;
let hasGraduated = false;

// 4. Undefined (no value assigned)
let notAssigned;
console.log(notAssigned); // undefined

// 5. Null (intentionally empty)
let emptyValue = null;

// 6. Object (complex data)
let person = {
    name: "Alice",
    age: 30,
    city: "New York"
};

// 7. Array (list of values)
let colors = ["red", "green", "blue"];
let numbers = [1, 2, 3, 4, 5];

// Checking data types
console.log(typeof firstName);  // "string"
console.log(typeof age);        // "number"
console.log(typeof isStudent);  // "boolean"
