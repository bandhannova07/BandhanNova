Chapter 1: HTML Basics

Learn the fundamental building blocks of web pages. HTML (HyperText Markup Language) is the standard markup language for creating web pages.

📘 Introduction

HTML is the backbone of every website on the internet. It provides structure and meaning to web content through a system of elements and tags. Think of HTML as the skeleton of a house - it provides the framework that everything else is built upon.

Every webpage you visit, from simple blogs to complex applications like Facebook or Google, starts with HTML. It tells the browser what content to display and how to organize it, whether it's headings, paragraphs, images, or links.

In this chapter, you'll learn how HTML elements work, how to create your first webpage, and understand the basic structure that makes the web possible.

🖥️ Code Example 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is my first paragraph in HTML!</p>
    <a href="https://www.example.com">Visit Example.com</a>
</body>
</html>

Chapter 2: HTML Elements & Tags

Dive deeper into HTML elements and learn about different types of tags, their purposes, and how to use them effectively.

📘 Introduction

HTML elements are the building blocks of web pages. Each element serves a specific purpose and provides semantic meaning to your content. Understanding different types of elements is crucial for creating well-structured, accessible websites.

Elements can be block-level (taking full width) or inline (flowing with text). Some elements are self-closing, while others require both opening and closing tags. Each element can also have attributes that provide additional information or functionality.

In this chapter, we'll explore the most commonly used HTML elements and learn when and how to use them properly.

Chapter 1: Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a series of elements, which tell the browser how to display the content.

HTML code is interpreted by browsers to create structured web pages

Did You Know?

HTML was first created by Tim Berners-Lee in 1990 as a way to share documents between researchers. Today, it's the foundation of every website on the internet!

HTML Document Structure

Every HTML document follows a basic structure. Let's look at the essential components:

Basic HTML Structure 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is my first paragraph.</p>
</body>
</html>

Understanding HTML Elements

HTML elements are the building blocks of web pages. Each element consists of:

Every HTML element has an opening tag, content, and closing tag

  • Opening Tag: Marks the beginning of an element (e.g., <h1>)
  • Content: The text or other elements inside
  • Closing Tag: Marks the end of an element (e.g., </h1>)
