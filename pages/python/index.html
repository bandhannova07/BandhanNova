Introduction to Python

Discover the Power of Python Programming

What is Python?

Python is a high-level, interpreted programming language that has revolutionized the world of software development. Created by Guido van Rossum in 1991, Python was designed with a philosophy that emphasizes code readability and simplicity. Its clean, English-like syntax makes it an ideal first programming language for beginners, while its powerful capabilities make it a favorite among experienced developers.

What sets Python apart is its versatility and ease of use. Unlike many programming languages that require complex syntax and lengthy code, Python allows you to accomplish more with fewer lines of code. This "batteries included" philosophy means Python comes with a comprehensive standard library that provides tools for everything from web development to data analysis.

Today, Python powers some of the world's most popular applications and services. Companies like Google, Netflix, Instagram, and Spotify rely heavily on Python for their core operations. Whether you're interested in building websites, analyzing data, creating artificial intelligence, or automating repetitive tasks, Python provides the tools and simplicity to bring your ideas to life.

Your First Python Program

hello-world.py 
# Welcome to Python Programming!
# This is a comment - it explains what the code does

# Print a simple message
print("Hello, World!")

# Store information in variables
student_name = "Alex"
course_name = "Python Programming"
current_year = 2024

# Use variables in strings with f-strings
welcome_message = f"Hi {student_name}! Welcome to {course_name} in {current_year}."
print(welcome_message)

# Simple calculations
age = 25
birth_year = current_year - age
print(f"If you're {age} years old, you were born in {birth_year}")

# Working with lists
favorite_languages = ["Python", "JavaScript", "Java"]
print(f"Popular programming languages: {favorite_languages}")

# Simple function
def greet_student(name):
    return f"Hello {name}, ready to learn Python?"

# Call the function
greeting = greet_student("Future Developer")
print(greeting)

Understanding Variables in Python

Variables in Python are like labeled containers that store data values. Think of them as boxes with names written on them - you can put different types of information inside and refer to them by their names later. Unlike some programming languages, Python doesn't require you to declare what type of data a variable will hold; it figures this out automatically based on the value you assign.

Python's approach to variables is both flexible and intuitive. You can create a variable simply by assigning a value to a name using the equals sign (=). The variable immediately takes on the type of the data you've assigned to it, and you can even change the type later by assigning a different kind of value.

This dynamic typing system makes Python incredibly beginner-friendly while still being powerful enough for complex applications. Understanding how to work with variables and data types is fundamental to becoming proficient in Python programming, as they form the building blocks of all Python programs.

Python Variables and Data Types

variables-and-types.py 
# Python Data Types and Variables

# 1. Strings (text data)
name = "Sarah Johnson"
city = 'New York'  # Single or double quotes work
message = f"Hello, my name is {name} and I live in {city}"

# 2. Numbers
age = 28                    # Integer
height = 5.7               # Float (decimal)
temperature = -15.5        # Negative numbers

# 3. Boolean (True/False)
is_student = True
has_job = False
is_learning_python = True

# 4. Lists (ordered collections)
favorite_colors = ["blue", "green", "purple"]
numbers = [1, 2, 3, 4, 5]
mixed_list = ["Python", 2024, True, 3.14]

# 5. Dictionaries (key-value pairs)
person = {
    "name": "Alex",
    "age": 25,
    "city": "San Francisco",
    "skills": ["Python", "JavaScript", "SQL"]
}

# 6. None (represents absence of value)
middle_name = None

# Checking data types
print(f"Type of name: {type(name)}")
print(f"Type of age: {type(age)}")
print(f"Type of height: {type(height)}")
print(f"Type of is_student: {type(is_student)}")
print(f"Type of favorite_colors: {type(favorite_colors)}")
print(f"Type of person: {type(person)}")

# Variable reassignment (dynamic typing)
score = 95        # Integer
print(f"Score: {score} (type: {type(score)})")

score = "Excellent"  # Now it's a string
print(f"Score: {score} (type: {type(score)})")

# Working with variables
full_name = name.upper()
birth_year = 2024 - age
favorite_colors.append("red")
person["country"] = "USA"

print(f"Full name in caps: {full_name}")
print(f"Birth year: {birth_year}")
print(f"Updated colors: {favorite_colors}")
print(f"Person info: {person}")
