class User < ApplicationRecord
  has_many :posts, dependent: :destroy
  validates :email, presence: true, uniqueness: true
  validates :name, presence: true, length: { minimum: 2 }
  
  def full_name
    "#{first_name} #{last_name}"
  end
  
  def recent_posts(limit = 5)
    posts.order(created_at: :desc).limit(limit)
  end
end

class PostsController < ApplicationController
  before_action :authenticate_user!
  
  def index
    @posts = current_user.posts.includes(:user)
    render json: @posts, status: :ok
  end
  
  def create
    @post = current_user.posts.build(post_params)
    
    if @post.save
      render json: @post, status: :created
    else
      render json: { errors: @post.errors }, status: :unprocessable_entity
    end
  end
  
  private
  
  def post_params
    params.require(:post).permit(:title, :content, :published)
  end
end

Introduction to Ruby

Elegant Programming for Developer Happiness

What is Ruby?

Ruby is a dynamic, object-oriented programming language created by Yukihiro "Matz" Matsumoto in 1995 with a simple philosophy: make programming enjoyable and productive. Ruby is designed to be intuitive and expressive, allowing developers to write code that reads almost like natural English. This focus on developer happiness and productivity has made Ruby a favorite among programmers worldwide.

What makes Ruby special is its principle of "least surprise" - the language behaves in a way that minimizes confusion for experienced users. Ruby treats everything as an object, providing a consistent and powerful programming model. Its flexible syntax allows for multiple ways to accomplish the same task, giving developers the freedom to express their ideas naturally and elegantly.

Ruby gained massive popularity with the introduction of Ruby on Rails in 2004, a web application framework that revolutionized web development with its "convention over configuration" philosophy. Today, Ruby powers major websites like GitHub, Shopify, Airbnb, and Basecamp. Its combination of simplicity, power, and the robust Rails ecosystem makes it an excellent choice for web development, automation scripts, and rapid prototyping.

Your First Ruby Program

welcome_to_ruby.rb 
# Welcome to Ruby Programming!
# Ruby is designed for programmer happiness

# Simple variable assignment
name = "Ruby Developer"
age = 25
is_learning = true
favorite_languages = ["Ruby", "JavaScript", "Python"]

# Ruby's elegant string interpolation
puts "Hello, #{name}!"
puts "You are #{age} years old."

# Everything is an object in Ruby
puts "Ruby".class          # String
puts 42.class             # Integer
puts true.class           # TrueClass

# Method definitions are clean and simple
def greet_programmer(name, language = "Ruby")
  "Welcome to #{language}, #{name}! Let's build something amazing!"
end

# Calling methods
greeting = greet_programmer(name)
puts greeting

# Ruby's expressive conditionals
if is_learning
  puts "Keep coding! You're doing great!"
else
  puts "Time to start your Ruby journey!"
end

# Elegant iteration with blocks
puts "\nYour favorite languages:"
favorite_languages.each do |language|
  puts "- #{language} is awesome!"
end

# Ruby's powerful enumerable methods
numbers = [1, 2, 3, 4, 5]
squared = numbers.map { |n| n * n }
puts "\nOriginal: #{numbers}"
puts "Squared: #{squared}"

# Object-oriented programming made simple
class Developer
  attr_accessor :name, :experience
  
  def initialize(name, experience = 0)
    @name = name
    @experience = experience
  end
  
  def introduce
    "Hi! I'm #{@name}, a developer with #{@experience} years of experience."
  end
  
  def level_up!
    @experience += 1
    puts "#{@name} gained experience! Now at #{@experience} years."
  end
end

# Creating and using objects
dev = Developer.new("Alice", 2)
puts dev.introduce
dev.level_up!

# Ruby's beautiful syntax for common tasks
(1..5).each { |i| print "#{i} " }
puts "\n"

# Hash (dictionary) creation and access
person = {
  name: "Bob",
  age: 30,
  city: "San Francisco",
  skills: ["Ruby", "Rails", "JavaScript"]
}

puts "#{person[:name]} lives in #{person[:city]}"
puts "Skills: #{person[:skills].join(', ')}"

Why Choose Ruby?

Ruby powers popular applications like GitHub, Shopify, Airbnb, and Twitter. Its elegant syntax and powerful Rails framework make it ideal for web development.

Ruby Philosophy

Ruby follows several key principles that make it unique:

  • Programmer Happiness: Code should be enjoyable to write and read
  • Principle of Least Surprise: Code should behave as you expect
  • Everything is an Object: Pure object-oriented design
  • Expressive Syntax: Natural, English-like code structure
