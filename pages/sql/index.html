Introduction to Databases

Understanding Data Storage and Management

What is a Database?

A database is a structured collection of data that is stored and organized in a way that allows for efficient retrieval, management, and manipulation. Think of it as a digital filing cabinet where information is systematically organized into tables, rows, and columns. Databases are the backbone of virtually every modern application, from social media platforms to banking systems, e-commerce websites to healthcare records.

SQL (Structured Query Language) is the standard language for communicating with relational databases. It allows you to create, read, update, and delete data using simple, English-like commands. SQL is incredibly powerful yet intuitive - you can ask complex questions about your data and get precise answers in milliseconds, even when dealing with millions of records.

Understanding databases and SQL is crucial in today's data-driven world. Whether you're a web developer building user authentication systems, a data analyst uncovering business insights, or an entrepreneur managing customer information, SQL skills are essential. Major companies like Google, Amazon, Facebook, and Netflix rely heavily on databases to store and analyze the vast amounts of data they process daily.

Your First SQL Queries

basic-queries.sql 
-- Welcome to SQL! Let's explore a sample database

-- Create a simple users table
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(150) UNIQUE NOT NULL,
    age INT,
    city VARCHAR(50),
    registration_date DATE DEFAULT CURRENT_DATE
);

-- Insert sample data
INSERT INTO users (name, email, age, city) VALUES
('Alice Johnson', 'alice@email.com', 28, 'New York'),
('Bob Smith', 'bob@email.com', 34, 'Los Angeles'),
('Carol Davis', 'carol@email.com', 22, 'Chicago'),
('David Wilson', 'david@email.com', 31, 'Houston'),
('Emma Brown', 'emma@email.com', 26, 'Phoenix');

-- Basic SELECT queries
-- Get all users
SELECT * FROM users;

-- Get specific columns
SELECT name, email FROM users;

-- Filter data with WHERE clause
SELECT name, age FROM users 
WHERE age > 25;

-- Sort data
SELECT name, city FROM users 
ORDER BY age DESC;

-- Count records
SELECT COUNT(*) as total_users FROM users;

-- Filter by city
SELECT name, email FROM users 
WHERE city = 'New York';

-- Find users with specific age range
SELECT name, age, city FROM users 
WHERE age BETWEEN 25 AND 30;

-- Search for names containing specific text
SELECT * FROM users 
WHERE name LIKE '%Johnson%';

-- Update user information
UPDATE users 
SET city = 'San Francisco' 
WHERE name = 'Alice Johnson';

-- Delete a user
DELETE FROM users 
WHERE email = 'bob@email.com';

Databases store structured data in tables and respond to SQL queries

Why Learn SQL?

SQL is the standard language for managing relational databases. It's used by companies like Google, Facebook, Netflix, and virtually every tech company that handles data.

Types of Databases

There are several types of databases, each designed for specific use cases:

  • Relational Databases: Store data in tables with rows and columns (MySQL, PostgreSQL, SQLite)
  • NoSQL Databases: Store data in flexible, non-tabular formats (MongoDB, Redis)
  • Graph Databases: Store data as nodes and relationships (Neo4j, Amazon Neptune)
  • Columnar Databases: Store data in columns for analytics (Amazon Redshift, BigQuery)

Your First SQL Query

Basic SELECT Query 
-- Select all columns from the users table
SELECT * FROM users;

-- Select specific columns
SELECT name, email, created_at FROM users;

SELECT name, email 
FROM users 
WHERE age >= 18;

-- Count total users
SELECT COUNT(*) as total_users FROM users;

-- Select users and order by name
SELECT name, email, age
FROM users
WHERE age >= 21
ORDER BY name ASC;
