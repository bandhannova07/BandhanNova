class Student {
private:
    std::string name;
    int age;
    std::vector grades;

public:
    Student(const std::string& n, int a) : name(n), age(a) {}
    
    void addGrade(double grade) {
        grades.push_back(grade);
    }
    
    double getAverage() const {
        if (grades.empty()) return 0.0;
        double sum = 0.0;
        for (double grade : grades) {
            sum += grade;
        }
        return sum / grades.size();
    }
};

Introduction to C++

Master the Foundation of Systems Programming

What is C++?

C++ is a powerful, general-purpose programming language that revolutionized software development by combining the efficiency of C with object-oriented programming capabilities. Created by Bjarne Stroustrup at Bell Labs in 1979, C++ was designed to provide programmers with fine-grained control over system resources while offering high-level programming constructs for building complex applications.

What makes C++ unique is its multi-paradigm approach - it supports procedural, object-oriented, and generic programming styles. This flexibility allows developers to choose the most appropriate programming paradigm for their specific problem domain. C++ is compiled to native machine code, resulting in highly efficient executables that can run with minimal runtime overhead.

Today, C++ powers critical systems across industries - from operating systems like Windows and Linux, to game engines like Unreal Engine, to high-frequency trading systems on Wall Street. Major companies like Google, Microsoft, Adobe, and Tesla rely on C++ for performance-critical applications where every millisecond matters.

using namespace std;

int main() {
    // Output welcome messages
    cout << "Hello, World!" << endl;
    cout << "Welcome to C++ Programming!" << endl;
    
    // Variables and data types
    string studentName = "Alex";
    string courseName = "C++ Programming";
    int currentYear = 2024;
    int age = 25;
    double gpa = 3.85;
    bool isEnrolled = true;
    
    // Using variables in output
    cout << "Student: " << studentName << endl;
    cout << "Course: " << courseName << " (" << currentYear << ")" << endl;
    cout << "Age: " << age << " years old" << endl;
    cout << "GPA: " << gpa << endl;
    cout << "Enrolled: " << (isEnrolled ? "Yes" : "No") << endl;
    
    // Simple calculations
    int birthYear = currentYear - age;
    cout << "Birth year: " << birthYear << endl;
    
    return 0;
}

C++ Features

C++ combines the best of both worlds - low-level control and high-level features:

C++ provides direct control over memory allocation and management

  • High Performance: Direct memory management and efficient compilation
  • Object-Oriented: Classes, inheritance, polymorphism, and encapsulation
  • Memory Control: Manual memory management with pointers and references
  • Generic Programming: Templates and STL for reusable code
