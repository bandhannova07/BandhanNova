class Student {
private:
    std::string name;
    int age;
    std::vector grades;

public:
    Student(const std::string& n, int a) : name(n), age(a) {}
    
    void addGrade(double grade) {
        grades.push_back(grade);
    }
    
    double getAverage() const {
        if (grades.empty()) return 0.0;
        double sum = 0.0;
        for (double grade : grades) {
            sum += grade;
        }
        return sum / grades.size();
    }
};

What is C++?

C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language. It's known for its performance, flexibility, and is widely used in system programming, game development, embedded systems, and high-performance applications.

C++ excels in performance-critical applications across multiple domains

Why Learn C++?

C++ powers operating systems, game engines, databases, and performance-critical applications. Companies like Google, Microsoft, and Adobe rely heavily on C++.

C++ Features

C++ combines the best of both worlds - low-level control and high-level features:

C++ provides direct control over memory allocation and management

  • High Performance: Direct memory management and efficient compilation
  • Object-Oriented: Classes, inheritance, polymorphism, and encapsulation
  • Memory Control: Manual memory management with pointers and references
  • Generic Programming: Templates and STL for reusable code
