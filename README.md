## 🛠️ Installation & Setup

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A web server (optional for local development)
- Git (for cloning the repository)

### Method 1: Direct Download

1. **Download the project files**
   ```bash
   git clone https://github.com/bandhannova07/BandhanNova.git
   cd BandhanNova
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local web server for better experience

### Method 2: Local Web Server

#### Using Python (Recommended)

```bash
# Navigate to project directory
cd BandhanNova

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Open browser to http://localhost:8000
```

#### Using Node.js (http-server)

```bash
# Install http-server globally
npm install -g http-server

# Navigate to project directory
cd BandhanNova

# Start server
http-server

# Open browser to http://localhost:8080
```

#### Using PHP

```bash
# Navigate to project directory
cd BandhanNova

# Start PHP server
php -S localhost:8000

# Open browser to http://localhost:8000
```

#### Using Live Server (VS Code Extension)

1. Install "Live Server" extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

## 🌐 Deployment

### Netlify (Recommended)

1. **Fork/Clone this repository**
2. **Connect to Netlify**
   - Go to [Netlify](https://netlify.com)
   - Click "New site from Git"
   - Connect your GitHub repository
   - Deploy settings:
     - Build command: (leave empty)
     - Publish directory: `/` (root)

3. **Custom Domain (Optional)**
   - Add your custom domain in Netlify settings
   - Update URLs in the code to match your domain

### GitHub Pages

1. **Enable GitHub Pages**
   - Go to repository Settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder

2. **Access your site**
   - Your site will be available at: `https://yourusername.github.io/BandhanNova`

### Vercel

1. **Import Project**
   - Go to [Vercel](https://vercel.com)
   - Click "New Project"
   - Import from GitHub
   - Deploy with default settings

## 🔧 Customization

### Updating Company Information

1. **Edit `index.html`**
   - Update company name, description, and contact details
   - Modify structured data in the `<script type="application/ld+json">` section

2. **Update Contact Information**
   - Edit `contact.html` to update contact details
   - Modify social media links

3. **SEO Optimization**
   - Update meta tags in all HTML files
   - Modify `sitemap.xml` with your domain
   - Update `robots.txt` if needed

### Adding New Pages

1. **Create new HTML file** (e.g., `services.html`)
2. **Follow the existing structure** from other pages
3. **Update navigation** in all files to include new page
4. **Add to sitemap.xml** for SEO

### Styling Changes

- All CSS is embedded in HTML files for simplicity
- Modify the `<style>` sections to customize appearance
- Colors, fonts, and layouts can be easily adjusted
